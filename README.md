# Ukrainian–German Cognates Project

This repository contains materials for the cross-linguistic analysis of Ukrainian–German cognates.  
The project focuses on semantic, morphological, and etymological correspondences between cognate pairs in both languages.

## Contents
- Frequency tables
- Cognate lists
- Analytical notes
  

## Purpose
The repository is designed to support students and researchers working on:
- cross-linguistic comparison,
- cognate identification,
- semantic classification,
- corpus-based frequency analysis.

## How to use
You may download or clone the repository and work with the provided datasets.  
Students in the course can use the shared Excel files to fill in frequency data collaboratively.
